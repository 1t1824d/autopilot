# autopilot

我在自动驾驶开发过程中的一些经验总结，欢迎交流和学习 ~

## 本地启动服务

暂时只支持本地预览效果，线上地址等我有空弄一个

```bash
git clone https://github.com/GitHubJackson/autopilot.git
cd autopilot
pnpm install
pnpm run dev
```
